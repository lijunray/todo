# Todo List
Simplest Todo List App.
### Installation
Make sure you have node.js.
```
$ npm install
```
or
```
$ npm i
```

### Build(Watch)
```
$ npm run watch
```
or 
```
$ webpack webpack.config.js
```

See webpack generates a `app.bundle.js` under public, then open index.html in your browser to see the app.
